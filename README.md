# DesignChecker API

## Instalação

Para usar o DesignChecker em um projeto Maven, edite o arquivo `pom.xml`:

```xml
<project ...>

	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>

    <dependencies>
		<dependency>
		    <groupId>com.github.felipeborgez</groupId>
		    <artifactId>designchecker</artifactId>
		    <version>0.1</version>
		</dependency>
    </dependencies>
</project>
```
